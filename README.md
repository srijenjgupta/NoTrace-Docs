# 🍃 ZenDocs Suite
**A Zero-Retention, High-Performance Document Processing Studio**

ZenDocs is a privacy-first, full-stack web application designed for seamless document manipulation. Built with a focus on "Zero-Retention" security, every file uploaded is processed in RAM/Temporary storage and wiped immediately after the session.

## 🚀 Live Demo
https://srijenjgupta.github.io/NoTrace-Docs/

## ✨ Key Features
* **Interactive PDF Studio:** A visual workspace to drag, drop, reorder, and split PDF pages using `pdf.js`.
* **Smart Conversions:** High-fidelity conversion between PDF, Word, and Excel.
* **Advanced Compression:** Intelligent file size reduction for PDFs and Images.
* **Excel Security Unlocker:** Systematic removal of sheet protection and write-locks.
* **Privacy by Design:** No database, no cookies, and instant server-side file cleanup.

## 🛠️ The Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (SortableJS, PDF.js) - Hosted on **GitHub Pages**.
* **Backend:** Python (FastAPI), PyMuPDF, Pandas, OpenPyxl - Hosted on **Hugging Face Spaces**.
* **Engine:** Headless LibreOffice for enterprise-grade document rendering.
* **Deployment:** Docker-containerized backend for 16GB RAM scalability.

## 🧠 Product Vision (by Srijen Gupta)
As an Product enthusiast, I built ZenDocs to solve a common corporate friction point: the need for powerful document tools without compromising sensitive data to 3rd-party servers.
